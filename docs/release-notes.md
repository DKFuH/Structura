# Structura 0.1.0 Release Notes

## Summary

`0.1.0` is the first open-source release of Structura. The release establishes the core desktop workflow for managing chapter-based non-fiction book projects on Windows while keeping manuscript files, notes, and exports in a normal folder structure.

## Highlights

- create and open real project folders
- manage chapters and divider sections
- reorder chapters with drag and drop
- keep notes as Markdown files
- preview extracted text from `.docx` chapter files
- open chapters in external editors
- detect Word, LibreOffice, and TextMaker when available
- store app-level settings such as office path overrides
- configure clipboard-based workflow buttons for external review tools
- export a combined master document

## Known limitations

- Structura is not a DOCX editor
- the text preview is content-focused, not layout-accurate
- PDF generation is optional and depends on external tooling where available
- settings and workflow customization are intentionally lightweight in `0.1.0`
- screenshots and polished release packaging are still pending follow-up work

## Intended audience for 0.1.0

This release is aimed at early adopters, contributors, and testers who want a local-first manuscript organizer and are comfortable building or evaluating an early Lazarus desktop application.
